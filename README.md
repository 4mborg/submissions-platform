# submissions-platform

This is the scaffolding for the project.

## Getting Starting

This project requires NodeJS and NPM and these instructions assume you have them installed and configured.

Run the following commands to prepare the environment:

```bash
npm install
```

Then for rapid development:

```bash
npm run dev
```

To run in production mode:

```bash
npm run start
```

## Docker/Devcontainer fans

Since this project uses raw nodejs and npm you can use the node images as provided by docker hub.

Further instructions will be added once this has been fleshed out.
